# Windows

Amethyst offers installers for x64 and Arm based Windows systems. The x64 version works on Windows 10 and 11 while the Arm version only works on Windows 11 Arm.

The installers can be found in the [releases](https://github.com/kinderhead/Amethyst/releases).

:::note
Daemon mode has some limitations on Windows. See the [handbook](../../handbook/daemon.mdx) for more information.
:::
